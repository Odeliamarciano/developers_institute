  • The h1 --> h6 define different size of the headings h1 was the bigger one and h6 the small one

  • The div tag defines a division or a section in an HTML document..

  • The ol tag defines an Unordered List

  • The ul tag defines an Ordered List

  • The li tag defines a list item.

  • The br tag inserts a single line break.

  • The img tag defines an image in an HTML page.

  • The a tag defines an Hyperlinks in an HTML page.
